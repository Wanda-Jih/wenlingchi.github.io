Automation Test for SHOPBACK

This automated test is suitable for automated testing of login and registration of the SHOPBACK android platform.

  • Project Period: Jan. 2021 - Mar. 2021
  • Project Duration: Three months
  • Project Type: Individual Project

Introduction

This project is for automatically doing signin and login flow at SHOPBACK android platform. For QA engineer at SHOPBACK, login and signup is a routine work. Thus, this project can save us a lot of time.

At first glance, this may be a very simple automated procedure. However, the registration process of SHOPBACK is very complicated. In addition to checking whether a referral code is used, it is also necessary to use a mobile phone to register. Therefore, before starting to run the automated program, I must first create a phone that can skip OTP in the database. After the registration process is completed, I need to go to the database to check whether the newly created account uses referral code.
